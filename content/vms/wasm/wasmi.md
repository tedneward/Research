[Source](https://github.com/paritytech/wasmi)

Implemented in Rust. Interpreted.

**Non-MVP features supported**: 

- Mutable Globals
- Sign-extension operators
- Non-trapping conversions
- Multi-Value
- Bulk Memory Operations
- Reference Types
- Tail Calls
- Extended Const

**Non-web standards**

- WASI

**Used by**

- Substrate
- smoldot
- Ayaka Game Engine
