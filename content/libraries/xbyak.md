~~~~~~

[Source](https://github.com/herumi/xbyak)

Xbyak is a C++ header-only library that enables dynamic assembly of x86/x64 instructions using mnemonics.

### Derived Projects

-   [Xbyak\_aarch64](https://github.com/fujitsu/xbyak_aarch64/) : for AArch64
-   [Xbyak\_riscv](https://github.com/herumi/xbyak_riscv) : for RISC-V
