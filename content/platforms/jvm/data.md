## Relational (JDBC)
[JDBC 4.1 Specification (JSR 221)](https://download.oracle.com/otn-pub/jcp/jdbc-4_1-mrel-spec/jdbc4.1-fr-spec.pdf)

[JDBC 3.0 Specification (JSR 54)](http://java.cnam.fr/iagl/biblio/spec/jdbc-3_0-fr-spec.pdf) (Historical interest only; everything should be up to 4.x)

[HyperSQL](/storage/hsqldb)

[Apache Derby](http://db.apache.org/derby/) is an RDBMS written in Java; it can host inside JVM programs, it can host JVM code inside of it, as well as operate in a traditional client-server model.

[P6Spy](https://github.com/p6spy/p6spy): a framework that enables database calls to be seamlessly intercepted and logged with no code changes to the application.
