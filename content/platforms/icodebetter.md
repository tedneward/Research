Tables (rows/columns).

### User interface builder
Components are the building blocks of Budibase. They are used to construct your user interface. Within Budibase, there is a growing list of components at your disposal.

* Container
* Repeater
* Forms
* Table
* Chart
* Card
* Text
* Button
* Icon
* Image
* Embed
