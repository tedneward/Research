[sick.codes](https://github.com/sickcodes): collection of interesting macOS-related repositories

    - [Docker-OSX](https://github.com/sickcodes/Docker-OSX): Run macOS VM in a Docker! Run near native OSX-KVM in Docker! X11 Forwarding! CI/CD for OS X Security Research! Docker mac Containers.
    - [Docker-eyeOS](https://github.com/sickcodes/Docker-eyeOS): Run iPhone (xnu-arm64) in a Docker container! Supports KVM + iOS kernel debugging (GDB)! Run xnu-qemu-arm64 in Docker! Works on ANY device.
    - [osx-optimizer](https://github.com/sickcodes/osx-optimizer): OSX Optimizer: Optimize MacOS - Shell scripts to speed up your mac boot time, accelerate loading, and prevent unnecessary throttling.
