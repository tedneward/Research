## Articles

[90 Days of DevOps](https://github.com/MichaelCade/90DaysOfDevOps): "This repository is my documenting repository for learning the world of DevOps. I started this journey on the 1st January 2022 and I plan to run to March 31st for a complete 90-day romp on spending an hour a day including weekends to get a foundational knowledge across a lot of different areas that make up DevOps." Including:

* What is and why do we use DevOps
* Learning a Programming Language
* Knowing Linux Basics
* Understand Networking
* Stick to one Cloud Provider (he uses Azure)
* Use Git Effectively
* Containers (Docker)
* Kubernetes
* Learn Infrastructure as Code (Terraform)
* Automate Configuration Management (Ansible)
* Create CI/CD Pipelines (Jenkins)
* Monitoring, Log Management, and Data Visualization
* Store & Protect Your Data
