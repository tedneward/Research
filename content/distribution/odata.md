## Clients/SDKs

[Odata.NET](https://github.com/OData/odata.net): .NET libraries and frameworks

[ODataConnectedService](https://github.com/OData/ODataConnectedService): A Visual Studio extension for generating client code for OData Services

[Odata WebAPI](https://github.com/OData/WebApi): A server library built upon ODataLib and WebApi

[Apache Olingo](https://olingo.apache.org/): Java (v2, v4), Javascript (v4)

[RESTier](http://odata.github.io/RESTier/): RESTful API development framework for building standardized, OData V4 based RESTful services on .NET platform. [Github](https://github.com/OData/RESTier)

[Ecosystem](https://www.odata.org/ecosystem/): [VSCode extension](https://marketplace.visualstudio.com/items?itemName=stansw.vscode-odata)
