Offers a "signup"--maybe cloud-hosted service?

Feature list:
* Connects to any existing SQL database (MySQL, Postgres, MS SQL)
* Spreadsheet UI
* Views: Grid, Gallery, Kanban, Calendar, Custom
* Send Notifications (Slack)
* APIs: REST, GraphQL
