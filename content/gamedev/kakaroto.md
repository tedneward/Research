title=kakaroto TTRPG utilities
tags=gamedev, place
summary=A collection of Roll20 and Foundry utilities.
~~~~~~

[Source](https://github.com/kakaroto)

- [Beyond20](https://beyond20.here-for-more.info/) : A Browser extension to integrate D&D Beyond character sheets into Roll 20 or Foundry VTT
- [Roll20 Exporter](https://github.com/kakaroto/R20Exporter): A script to export a Roll20 campaign to a ZIP file
- [Roll20 Converter](https://github.com/kakaroto/R20Converter) : A script to convert a Roll20 campaign into a pre-configured world for Foundry VTT
- [Roll20 Resources](https://github.com/kakaroto/Roll20_resources)
- [FVTT Modules](https://github.com/kakaroto?utf8=%E2%9C%93&tab=repositories&q=fvtt-module&type=&language=) : Various modules for Foundry VTT that improve Quality of Life or add some familiar features that Roll 20 had for those who don't like changing their habits.

    - [FVTT Module API](https://github.com/kakaroto/fvtt-module-api): This modules makes the Foudry VTT API accessible via HTTP.
    - [FVTT Module EntityOrder](https://github.com/kakaroto/fvtt-module-entityorder)
    - [FVTT Module ZOrder](https://github.com/kakaroto/fvtt-module-zorder)
    - [FVTT Module Infinite Folders](https://github.com/kakaroto/fvtt-module-infinite-folders)

- [foundry-vtt-silvaa-modules](https://github.com/kakaroto/foundry-vtt-sillvva-modules)
