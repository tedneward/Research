title=SimpleScript

summary=SimpleScript is a simple programming tool for scripting languages (Perl, PHP, Python, Ruby, TCL) written in Obj-C/Cocoa. It can be used by experienced developers to write and test on the fly small routines, and is a good solution for educational purposes. Do not use it in production environments or to manage large projects.
