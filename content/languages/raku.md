Raku documentation lists the following topics:
* Concurrency
* Inter-process communication
* Classes, objects, modules
* Operator overloading/creation
* Containers
* C-level FFI
* Compile-time traits
* Statement prefixes
* Phasers: Program execution phases and corresponding phaser blocks
* Metaobject protocol (MOP)

Rakudo is the compiler for the Raku language
https://github.com/rakudo/rakudo
A member of the Perl family of programming languages. Formerly known as Perl 6, it was renamed in October 2019. Raku introduces elements of many modern and historical languages. Compatibility with Perl was not a goal, though a compatibility mode is part of the specification.
