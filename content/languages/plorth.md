title=Plorth
tags=language, concatenative, web assembly
summary=Embeddable Forth inspired scripting language.
~~~~~~

[Source](https://github.com/RauliL/plorth)

A stack based, concatenative, strongly typed functional scripting language which is easy to embed to applications written in C++.

Features:

-   Strong typing.
-   Supports JSON data types and syntax.
-   Garbage collected.
-   Prototype based OO, just like in JavaScript.
