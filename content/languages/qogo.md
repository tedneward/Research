summary=Developed to teach 5 to 10 year olds the principles of programming via the venerable "Turtle Graphics" paradigm.
~~~~~~

[Source](http://qs.co.nz/Tcl/Qogo/)

It is written in TCL so it runs on all computer platforms. It features a full IDE with undo support and interactive construction as well as being able to save newly created commands as new instructions.
