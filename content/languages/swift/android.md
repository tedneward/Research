### Articles, Blogs, Essays

- [You Can Now Use Apple's Swift Coding Language to Build Android Apps](https://www.extremetech.com/mobile/you-can-now-use-apples-swift-coding-language-to-build-android-apps): "Apple's Swift programming language now officially [supports native Android app development](https://www.swift.org/blog/swift-6.3-released/). This new capability comes with Swift 6.3, which ships an officially maintained SDK and toolchain for Android. Developers can use these to write core app logic in Swift and run it on Android, as well as iOS, macOS, and other platforms."
