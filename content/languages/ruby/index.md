### Implementations:

* [Main](https://github.com/ruby/ruby): The original Ruby implementation, written in C.

    * Can be hosted in a C application; see [Running Ruby in C](https://silverhammermba.github.io/emberb/embed/).

* JVM: [JRuby](https://www.jruby.org/) ([Source](https://github.com/jruby/jruby)): Ruby implementation written for the JVM.
* CLR: IronRuby (appears to be stalled)
* C: [mRuby](https://github.com/mruby/mruby): A lightweight implementation of Ruby. Has a compile-time package manager.
* C: [mruby/c](https://github.com/mrubyc/mrubyc): An even smaller implementation of Ruby for microcontrollers with limited memory.
* JS: [Opal](../opal): A Ruby-to-JS transpiler.
* C: [tinyrb](https://github.com/macournoyer/tinyrb): A tiny subset of Ruby with a Lua'esc VM.

### Tools:

* [Sorbet](../sorbet): Type-checker for Ruby.
