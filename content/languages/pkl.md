- [`apple/pkl-vscode`](https://github.com/apple/pkl-vscode) - Pkl language support for VS Code
- [`apple/pkl.tmbundle`](https://github.com/apple/pkl.tmbundle) - TextMate bundle for Pkl
- [`apple/rules_pkl`](https://github.com/apple/rules_pkl) - Bazel build rules for Pkl
- [`apple/tree-sitter-pkl`](https://github.com/apple/tree-sitter-pkl) - Tree-sitter parser for Pkl
